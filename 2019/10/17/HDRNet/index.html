HDRNet

提炼HDRNet，参考http://simtalk.cn/2018/06/05/HDRNET/

核心思想

  1. 利用双边网格思想挑出所有非连续域，包括位置，强度，以使得低分辨率的字典图能包含高分辨率的大多数信息。正因如此 guidance map 才能根据字典图上采样得到高分辨的光照图
  2. 字典图储存的是各个通道的权重参数（*）以及偏移（+），用于“照亮”原图像；而 guidance map 的存在意义是生成更加灵活的强度，而不仅限于 RGB。
  3. local features 关注的是与空间相关的局部特征，可用局部直方图均衡化理解；global features 关注的是全局特征，比如室内室外等；如果去掉 global features 也会导致增强局部化，失去了协调性
