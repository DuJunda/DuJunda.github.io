MAML(模型无关的元学习)

提炼与阅读Model-Agnostic Meta-Learning for Fast Adaptation of Deep Networks，参考算法详解，代码

核心思想

简单来说，元学习解决的如何训练一个模型，使其面对一个新的任务时，能在 one/few-shot 快速收敛到一个较优的解。
本文通过多个任务的利用的 support set 得到预训练模型计算针对的 query set 的梯度图之和来更新原始模型，以鼓励更新后的模型得到多个任务的共有特征。
